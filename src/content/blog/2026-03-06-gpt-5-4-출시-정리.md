OpenAI가 2026년 3월 5일에 GPT-5.4를 공개했습니다.  
이번 발표를 보면 단순히 모델 성능 숫자를 올리는 것보다, 실제 업무 흐름에서 “덜 왔다 갔다 하게” 만드는 데 초점이 맞춰진 느낌이었습니다.

특히 코딩, 도구 연동, 웹 탐색, 문서 작업까지 하나로 묶어서 개선했다는 점이 이번 릴리스의 핵심으로 보입니다.

## 핵심 요약

- GPT-5.4는 ChatGPT, API, Codex에 동시에 공개됐고, 복잡한 업무를 위한 메인 모델 포지션으로 소개됐습니다.
- GPT-5.4 Pro도 같이 공개됐으며, 가장 어려운 작업에서 최대 성능이 필요한 사용자를 위한 라인업으로 분리됐습니다.
- API 기준 `gpt-5.4`, `gpt-5.4-pro`가 제공되고, ChatGPT에서는 GPT-5.4 Thinking이 기존 5.2 Thinking을 대체하기 시작했습니다.

## 기사 내용

1. OpenAI 설명 기준으로 GPT-5.4는 최근 추론, 코딩, 에이전트 워크플로우 개선을 하나의 메인 모델에 통합한 버전입니다.
2. 지식 업무 벤치마크(GDPval)에서는 83.0%를 기록했고, 문서·스프레드시트·프레젠테이션 같은 실무형 작업 개선을 강조했습니다.
3. 컴퓨터 사용 영역에서는 일반 목적 모델로는 처음으로 네이티브 컴퓨터 사용 기능을 넣었고, OSWorld-Verified에서 75.0%를 제시했습니다.
4. 코딩 쪽은 GPT-5.3-Codex 수준을 유지하거나 일부 영역에서 넘어섰다고 설명했고, Codex `/fast` 모드에서 최대 1.5배 빠른 토큰 속도도 언급됐습니다.
5. 도구 사용에서는 `tool search`를 새로 도입해 많은 툴 정의를 한 번에 프롬프트에 넣지 않아도 되게 바뀌었고, MCP Atlas 테스트 기준 토큰 사용량 47% 절감 수치가 공개됐습니다.
6. ChatGPT 측면에서는 긴 작업에서 응답 시작 전에 작업 계획을 먼저 보여주고, 응답 중간에 방향 수정이 가능한 흐름을 강조했습니다.
7. 가격은 API 기준으로 GPT-5.2 대비 상향됐고(`gpt-5.4` 입력 $2.50/M, 출력 $15/M), 대신 토큰 효율 개선으로 총 사용량을 줄이는 방향을 함께 제시했습니다.
8. GPT-5.2 Thinking은 유료 사용자 기준 레거시 모델로 3개월 유지 후 2026년 6월 5일 종료 예정이라고 안내됐습니다.

## 코멘트

- 이번 릴리스는 “한 번 더 똑똑해진 모델”이라기보다, 실제 업무 체인에서 바로 써먹기 쉽게 조정된 버전이라는 인상이 강했습니다.
- 특히 툴 검색과 컴퓨터 사용 쪽은 에이전트 자동화 파이프라인을 만드는 입장에서 체감 변화가 클 것 같습니다.
- 한국어 실사용 기준으로는, 긴 문맥 유지와 중간 방향 수정이 실제로 얼마나 안정적인지 직접 써보면서 확인해볼 생각입니다.
